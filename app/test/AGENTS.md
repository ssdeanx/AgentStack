# Test Pages

## Purpose

This folder contains **test pages** for development and experimentation with UI components and server actions. Used to validate component integrations before deploying to production routes.

## Files

| File | Purpose |
|------|---------|
| `page.tsx` | Main test page entry point |
| `form.tsx` | Form component testing |
| `action.ts` | Server actions for form handling |

## Usage

Access at `/test` during development to experiment with:

- AI Elements component integration
- Server action patterns
- Form handling with Mastra agents
- UI component combinations

## Related

- `app/`: Parent Next.js App Router directory
- `src/components/ai-elements/`: AI components to test
- `ui/`: Base components to test
