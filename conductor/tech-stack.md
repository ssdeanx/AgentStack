# Technology Stack

## Core Language & Runtime
- **TypeScript**: Primary programming language for type-safety and modern features.
- **Node.js**: Runtime environment (v20.9.0+ required).

## Frontend Architecture
- **Next.js 16**: App Router for server-side rendering and efficient routing.
- **React 19**: Latest React features for UI development.

## AI & Agent Orchestration
- **Mastra**: Multi-agent framework for building agent-driven applications.
- **AI SDK (Vercel)**: Unified interface for interacting with LLMs.

## UI & Styling
- **Tailwind CSS 4**: Utility-first CSS framework for rapid styling.
- **shadcn/ui**: Base UI component primitives.
- **Lucide React**: Icon library.
- **Framer Motion**: Animation library.

## Data & Storage
- **PostgreSQL / Supabase**: Primary relational database and backend-as-a-service.
- **Vector Stores**: Integration with PgVector, Qdrant, and LanceDB via Mastra.
- **Other Storages**: Support for MongoDB, LibSQL, and Cloudflare D1.

## Development & Quality Assurance
- **Vitest**: Unit and integration testing framework.
- **ESLint & Prettier**: Code linting and formatting.
- **TypeScript**: Static type checking.
